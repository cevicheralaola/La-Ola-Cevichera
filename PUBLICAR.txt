LA OLA — Publicar (resumen)

1) SUPABASE
   Abre el archivo:  laola-supabase-pegar.sql
   Copia SOLO el contenido de ese archivo (es puro SQL).
   Supabase → SQL Editor → New query → pegar → Run.
   No pegues lineas con simbolos raros ni titulos decorativos.

2) NETLIFY (variables en tu sitio)
   SUPABASE_URL = https://tu-ref.supabase.co
   SUPABASE_ANON_KEY = clave anon o publishable (Settings → API)
   (Si el build no las lee: puedes subir supabase-config.js con URL y clave al repo y redeploy.)

3) GIT
   Push al repo. Netlify despliega solo.

4) Abre la URL de Netlify en el navegador.
